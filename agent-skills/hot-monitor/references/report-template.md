# 热点简报模板

分析 JSON 完成后，用以下结构回复用户（Markdown）。**不要将原始采集 JSON 全文粘贴进简报**——只展示分析后的条目。

```markdown
# 🔥 AI 热点简报：{keyword}

> 采集时间：{analyzed_at} · 候选 {total} 条 · 有效 {genuine} 条 · 紧急 {urgent} 条

## ⚡ 紧急热点（≥80 分）

| 分数 | 来源 | 标题 | 摘要 |
|------|------|------|------|
| 85 | bing | ... | ... |

链接：[标题](url)

## 📌 重要热点（60–79 分）

（同上表格，或无则写「暂无」）

## 📋 一般动态（50–59 分，可选）

简要列表即可。

## 🚫 已过滤（is_genuine=false 或 <50 分）

- **[来源]** 标题 — 理由一句话

## 来源统计

| 来源 | 条数 |
|------|------|
| bing | 3 |
| ... | ... |

---
*由 hot-monitor Agent Skill 生成 · 采集脚本未使用 OpenRouter*
```

## 多关键词

对每个关键词分别运行 `collect.py`，分别分析，最后用「## 关键词：xxx」分节合并为一份简报。
